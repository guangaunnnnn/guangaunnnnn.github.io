星巴克APP : 新增外送功能改版設計

By Guan 16 April 2020

版權說明：上圖中ICON為台灣星巴克APP版權所屬，除外送ICON為中國星巴克APP版權所屬，其餘ICON、插圖為自行繪製。

產品背景

讓偶爾有喝咖啡習慣、且有申辦星巴克星享卡的用戶，能更快速選購星巴克商品。並且針對上班不方便外出購買咖啡、或是再開會前需購買咖啡飲品的上班族，提供此類服務需求。

用戶需求

  • 滿足預定功能

  • 了解訂單情況

  • 能搭配星享卡使用

  • 能了解線上貨店內最新優惠、活動

  • 備註特殊需求

  • 有訂單紀錄查看、再次下訂

首頁

  1. 加入目前所在地點定位、將原下方主選單(Tab Bar)的「門市」功能上移，顯示所在位置，方便配送。

  2. 在功能點選範圍欄：

    1)新增外送功能文字、ICON

    2)加入「優惠卷」張數顯示，增加用戶消費、使用率。

  3. 將原「帳戶」功能移至主選單（Tab Bar），取代較少使用的「門市」功能： 用戶大多使用Google搜尋服務尋找附近星巴克門市，並且直接搭配Google Map路線規劃即可抵達附近門市。

  4. 調整星禮程配色，並且將目前星星數和累積晉級星星數目版面劃分。

  5. 取消「連結」功能，原位置改成「商品」取代：使用者若要前往星巴克粉絲專頁、網站等，對外網頁，可以直接在首頁的公告版面新增。

  6. 將原本的浮動窗口「按此付款」，移至主選單「付款」功能，不受背景圖片干擾。 並且多了圓角和增加面積，方便使用者點擊。

帳戶

  1. 新增「咖啡值」功能，開啟外送功能可以累積到100%可兌換飲品。

  2. 「消費記錄」：

    1)區分成門市「店內消費」、「外送訂單」記錄。

    2)若無「外送訂單」點擊此區的空白處可以跳轉到「商品」頁，立即點單。

商品

  1. 原為向右滑動瀏覽商品，改為使用者以「選擇分類」下拉選單，快速找到想要的飲品項目。
  2. 可選擇圖片瀏覽或圖文並列(多了飲品介紹與價錢)顯示。 (圖片操作：選取與未選取狀態)
  3. 飲品圖片＋號icon可方便使用者快速選購下單。
  4. 右上角原icon搜尋改成購物車。

飲品下單詳情頁1：飲品調整（新增功能）

  1. 上方區塊：品名、商品分類名稱、飲品內文
  2. 飲品調整選單：顯示目前選取狀態、可調整項目、數量
  3. 新增購物車按鈕：顯示數目、價錢

飲品下單詳情頁2：提交訂單（新增功能）

  1. 購物車文字旁顯示選購數量

  2. 配送時間提醒 飲品下單詳情頁3：確認地址（新增功能）

    提交訂單後，跳出地址確認框，讓用戶進行再次確認。 飲品下單詳情頁4：確認付款（新增功能）

  3. 顯示預計送達時間

  4. 我的訂單：預設隱藏，使用者可下拉選單進行再次確認

  5. 可用優惠卷顯示

  6. 支付方式（隨行卡、Line Pay、Apply Pay）

  7. 其他(餐具提供等)

  8. 取消/支付按鈕

飲品下單詳情頁5：下單成功頁面（新增功能）

  1. 「商品」部分新增「目前訂單」

  2. 點擊「查看所有消費記錄」可跳轉至「我的」——「消費記錄」頁面

  3. 點擊「騎手努力送達中」可查看「騎手當前所在位置」

飲品下單詳情頁6：騎手當前所在位置（新增功能）

  1. 騎手位置、騎手與門市的距離顯示
  2. 商品送達時間顯示
  3. 聯絡騎手按扭

飲品下單詳情頁7：訂單完成頁（新增功能）

  1. 再次購買按鈕：方便使用者再次進行選購
  2. 訂單評價
  3. 顯示完成配送時間、訂單詳情

飲品下單詳情頁8：訂單評價（新增功能）

「星級評分」除了讓滿意程度可以量化外，比起直接填寫數字，圖像式的填答方式能讓填答變得更有趣。

  1. 整體訂單評價：滿意/不滿意
  2. 商品、包裝五顆星顯示
  3. 配送：好與不好
  4. 其他建議

配色

依循品牌視覺配色。
